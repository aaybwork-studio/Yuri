# Context: Phase 2.2 - Reshaping the Landing Page UI

This context document details the structural, styling, and architectural decisions made for the landing page redesign of the YURI e-commerce theme. It acts as a single source of truth for the subsequent research and planning phases.

---

## 1. Scope and Design Objectives

The goal is to restructure and restyle the landing page of the YURI store to match the high-end luxury editorial aesthetic shown in the reference images, while retaining YURI's brand colors (Deep Forest Green, Sage, Wabi-Sabi Brown, Warm Neutral Sand, and Soft Warm Whites).

The landing page layout is composed of the following sections:
1. **Persistent Header (Nav)**: A sticky header with `Monogram.png` centered. Options/links on the left, utility actions (Search, Account, Cart) on the right.
2. **Hero Section**: A slideshow taking up 2/3 of the viewport height on desktop, transitioning between hero images using a smooth autoplaying fade/slide animation.
3. **Featured Products**: A borderless "floating product grid" section with a horizontal scroll viewer to browse collections, featuring "Add to cart" buttons on the product cards.
4. **Browse Section**:
   - **Mirrored Row 1**: Text content on the left, full-width/height image on the right. Takes 100vh on desktop, responsive natural height on mobile.
   - **Mirrored Row 2**: Full-width/height image on the left, text content on the right. Takes 100vh on desktop, responsive natural height on mobile.
   - **Category Carousel**: A horizontal slider at the end of the Browse section featuring square cards for collection links (Skin Care, Hand & Body, etc.).
5. **About Us Section**: Split layouts with solid background text block on one side and a high-resolution brand image on the other. Takes 100vh on desktop, natural height on mobile.
6. **Footer Section**: A rounded custom banner style footer card based on the second reference image. Contains newsletter signup at the top right, a large centered green wordmark (`Wordmark-green.png`), and organized link columns below.

---

## 2. Key Decisions & Clarified Gaps

Based on feedback and design requirements, the following decisions are locked:

* **No New Store Setup**: All changes must be implemented within the existing Shopify Dawn theme setup in the `/Users/kura/Antigravity Projects/LILY` workspace.
* **Liquid Integration**: We will edit the existing homepage template and sections in the theme. We will build custom theme sections (e.g., `custom-hero`, `custom-featured-products`, `custom-browse-mirrored`, `custom-about-us`) so they can be selected, configured, and reordered inside the Shopify Theme Editor.
* **Autoplay Slideshow**: The Hero section will shuffle images using a smooth client-side autoplaying carousel/slideshow.
* **Responsive Height Rules**: Full-screen sections (Mirrored sections, About Us) will be exactly `100vh` on desktop to create a premium editorial feel, but will fallback to natural scroll heights on mobile to prevent any text truncation or clipping.
* **Wordmark and Logo Placement**:
  - Center logo in navigation is `Logos and Branding/Monogram.png`.
  - Brand mark in the footer is `Logos and Branding/Wordmark-green.png`.
  - All source images must be copied into the theme's `assets/` folder to be accessible via `asset_url`.

### B. Custom Liquid Sections
We will create or modify the following Liquid sections:
1. `sections/header.liquid`: Customize to support splitting the main menu to render options on the left and right of a centered monogram logo.
2. `sections/custom-hero.liquid` [NEW]: Implements a 2/3 viewport height slideshow with absolute-positioned text and pagination dots, utilizing CSS fades.
3. `sections/custom-featured-products.liquid` [NEW]: Implements floating cards without borders and horizontal scroll support using standard browser flex scroll hooks or lightweight script wrappers.
4. `sections/custom-browse-mirrored.liquid` [NEW]: Mirrored row wrapper rendering two panels (mirrored) plus a category carousel at the bottom.
5. `sections/custom-about-us.liquid` [NEW]: Double-panel block with styled background pane on one side and brand image on the other.
