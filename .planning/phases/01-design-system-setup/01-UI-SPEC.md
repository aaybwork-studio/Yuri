## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- **Page-level width padding**: 32px (2rem) on mobile / 80px (5rem) on desktop for `.page-width`
- **Hero/Banner padding**: 40px/48px on mobile, 64px/96px on desktop
- **Primary/Secondary buttons padding**: 12px (1.2rem) vertical, 30px (3rem) horizontal
- **Footer card padding**: 32px/64px on mobile, 48px/80px on desktop

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#eeecd3` | Base body background (`--color-brand-bg`), transparent cards, ivory surfaces |
| Secondary (30%) | `#403002` | Dark brown (`--color-brand-text`), primary text, primary button backgrounds, header/footer text, outline borders (at 10% opacity) |
| Accent (10%) | `#efbf04` | Gold (`--color-accent-gold`), interactive hover states, primary checkout CTA backgrounds, hero sliders, configurable footer block |
| Accent (10%) | `#cc1574` | Rose (`--color-accent-rose`), highlights, sale badges, checkout CTA accents, configurable footer block |
| Destructive | `#cc1574` | Warning states, destructive actions, checkout warnings |

Accent reserved for:
- **Gold (`#efbf04`)**: Primary interactive states, hover transitions, primary checkout actions, hero CTA highlights, and configurable footer card background.
- **Rose (`#cc1574`)**: Sale badges, product highlighting details, active selection indicators, and alternative footer card background.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | `add to cart` (lowercase, Space Grotesk, 0.1em letter-spacing) |
| Secondary CTA | `shop all` or `discover more` (lowercase, Space Grotesk, 0.1em letter-spacing) |
| Empty state heading | `your cart is currently empty` |
| Empty state body | `continue browsing to find your next ritual.` |
| Error state | `could not add item to cart. please try again.` |
| Destructive confirmation | `remove item`: `are you sure you want to remove this item from your cart?` |
| Sale Badge | `sale` |
| Sold Out Badge | `sold out` (subdued `#403002` at 60% opacity) |
