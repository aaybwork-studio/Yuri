## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding (minimum threshold) |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing (default section padding) |
| 4xl | 128px | Page-level spacing (maximum section padding) |

Exceptions:
- Dot separators in navbar use `0.8rem` (8px) margin.
- Page margins (desktop page-width) use `5rem` (50px) padding (from `entry.css`).
- Dynamic section spacing ranges from 24px to 128px (step 8px) configurable in Liquid schema.

## Phase-Specific Layout & Interaction Contracts

### 1. Minimalist Navbar Reform (`sections/header.liquid`)
- **Logo Position**: Position logo on the right side of the navbar (`logo-right`).
- **Navigation Links**: Left/middle aligned, lowercase, dot-separated (e.g. `home • about • products`).
- **Icon Group**: Group search, star (wishlist), and cart icons inline with links. No button outlines or vertical border separators.
- **Hover Transitions**: Remove the legacy double-text-slide translate hover animation on links. Use a static color transition for modern simplicity.

### 2. Full-Screen Slideshow Hero (`sections/custom-hero.liquid`)
- **Dimensions**: Hardcode container height to `100vh` / `100dvh` for a full-screen immersive display.
- **Text Alignment**: Support dynamic Left, Center, Right alignment for slide text overlay blocks.
- **Backdrop Overlay**: Add a schema range slider (0% to 90% opacity in steps of 10%) for a subtle dark background overlay.

### 3. Spacing & Border Controls (All Custom Sections)
- **Top/Bottom Padding**: Expose top/bottom padding controls in the schema ranging from 24px to 128px (step size 8px).
- **Section Dividers**: Add a toggle checkbox to show/hide a thin bottom border (`border-b zen-border` at 10% opacity of the text color).

### 4. Carousel/Grid & Splits (`sections/custom-featured-products.liquid`)
- **Layout Mode Toggle**: Add schema option to switch between a horizontal scroll carousel and a static responsive grid.
- **Desktop Columns**: Allow selecting 3 or 4 columns for desktop.
- **Split Header Option**: Support splitting layout: section title/description on the left (1/3 width), product carousel/grid on the right (2/3 width).
- **Contrast Cards**: Support solid dark background card container style (color `#403002` or `#2C3A2E`) to showcase light/white product packaging.
- **Mirrored Split Panel Ratios**: Support select splits (50/50, 60/40, 40/60) for mirrored panels (`sections/custom-browse-mirrored.liquid`).
